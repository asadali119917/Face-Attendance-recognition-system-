SYSTEM CAPABILITIES
🎯
OBJECT DETECTION
YOLOv8n detects persons in real-time with bounding boxes, confidence scores, and corner decorations on every frame.
🔲
SEGMENTATION
YOLOv8n-seg creates coloured instance masks per person. Offline fallback uses GrabCut segmentation.
🧬
FACE CLASSIFICATION
LBPH face recognizer identifies enrolled students. 128×128 face crops are stored as embeddings in a JSON database.
🛡️
ANTI-SPOOFING
LBP texture variance liveness detection. Real faces score >0.50. Printed photo / screen attempts are flagged.
😊
EMOTION & AGE
Brightness and aspect-ratio heuristics estimate emotion state and age group from each detected face crop.
🖥️
FUTURISTIC GUI
Cyberpunk-themed Tkinter GUI with live video, metric bars, detection log, registration form, and database view.
📂
STUDENT DATABASE
JSON-backed database with enrollment date, detection history, last-seen timestamp, and average confidence tracking.
⚡
OFFLINE FALLBACK
If YOLOv8 weights cannot download, the system automatically falls back to OpenCV Haar cascades. Zero crashes.
